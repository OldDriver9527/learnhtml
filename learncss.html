Cascading Style Sheets工作原理 1.浏览器加载html文档，对文档进行解析 2.解析文档时会形成DOM，浏览器将DOM渲染为文档树 3.解析文档时浏览器还会拉取html文档关联的资源，如图像，视频，css。当拉取css时，浏览器会对css进行解析。 浏览器根据选择器将样式应用于节点 4.页面在屏幕上展示

基本语法 selector{property:value} 多个属性使用;隔开 若属性值包含多个单词，属性值使用双引号标识

高级语法 选择器组 selector1,selector2{property:value} 选择器组中可以包含多个选择器，可以为多个选择器设置相同样式

样式层叠 样式来源有三种，浏览器为html定义的默认样式，用户定义的样式，开发者定义的样式 其中开发者定义的样式会覆盖用户定义的样式，用户定义的样式会覆盖浏览器默认样式

样式继承 为特定元素指定的样式不仅应用于该元素，还会应用于该元素的后代元素;并非所有属性都能继承

元素选择器 选择指定元素

派生选择器 包含后代选择器，子元素选择器，相邻兄弟选择器 后代选择器 A B 选择A元素后代中的所有B元素 子元素选择器 A>B 选择A元素子代中中所有B元素 相邻兄弟选择器 A+B 选择A元素后紧挨着的兄弟B元素

类名选择器 .className 选择具有指定类名的元素 id选择器 #id 选择具有指定id的元素 可以和派生选择器配合使用

属性选择器 [attribute] 选择具有指定属性的元素 [attribute=value] 选择具有指定属性，且属性值为指定值的元素 [attribute^=value] 选择具有指定属性，且属性值以value开头的元素 [attribute$=value] 选择具有指定属性，且属性值以value结尾的元素 [attribute*=value] 选择具有指定属性，且属性值包含value的元素 [attribute|=value] 选择具有指定属性，且属性值以独立单词value开头，value后可以是-或什么都没有 [attribute~=value] 选择具有指定属性，且属性值包含独立单词value的元素，value前后可以是空格

伪类 添加在选择器上的关键字，指定选择元素的特殊状态 selector:pseudo-class { property: value; } A:first-child 表示某元素第一个子元素的A元素

伪元素 添加在选择器上的关键字，对选择元素样式进行修改 selector:: pseudo-element { property: value; } ::before 在选择元素内容前添加内容 ::after 在选择元素内容后添加内容

通过多种选择器为同一元素指定样式，确定度高的选择器优先级更高

注释 CSS注释以/* 开始，以 */结束
